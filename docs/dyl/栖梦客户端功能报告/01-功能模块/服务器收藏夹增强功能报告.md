# QmClient 服务器收藏夹增强功能报告

## 一、功能概述

服务器收藏夹增强功能允许玩家为收藏的服务器添加备注、标签分组，并支持人数变化通知，提升服务器管理体验。

struct SFavoriteInfo

### 4.3 人数变化通知

#### 4.3.1 监控逻辑

```cpp
struct SServerMonitor
{
    NETADDR m_Addr;
    int m_LastPlayerCount;
    int m_Threshold;  // 变化阈值
    bool m_NotifyOnIncrease;
    bool m_NotifyOnDecrease;
};

void CheckServerChanges()
{
    for(auto &Monitor : m_vMonitors)
    {
        int CurrentCount = GetPlayerCount(Monitor.m_Addr);
        int Change = CurrentCount - Monitor.m_LastPlayerCount;
        
        if(abs(Change) >= Monitor.m_Threshold)
        {
            if(Change > 0 && Monitor.m_NotifyOnIncrease)
                ShowNotification(Monitor.m_Addr, Change);
            else if(Change < 0 && Monitor.m_NotifyOnDecrease)
                ShowNotification(Monitor.m_Addr, Change);
        }
        
        Monitor.m_LastPlayerCount = CurrentCount;
    }
}
```

#### 4.3.2 通知显示

```
┌─────────────────────────────────────┐

│ Players: 8 → 12 (+4)                │

└─────────────────────────────────────┘
```

### 4.4 配置变量

| 变量 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `cl_favorite_monitor` | INT | 0 | 启用收藏监控 |
| `cl_favorite_monitor_interval` | INT | 60000 | 检查间隔（毫秒） |
| `cl_favorite_notify_threshold` | INT | 3 | 通知阈值（人数变化） |
| `cl_favorite_notify_increase` | INT | 1 | 人数增加时通知 |
| `cl_favorite_notify_decrease` | INT | 0 | 人数减少时通知 |

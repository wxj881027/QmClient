# QmClient 持久化统计功能报告

## 一、功能概述

持久化统计功能将游戏统计数据保存到本地存储，支持跨局统计和图表可视化，让玩家追踪长期游戏表现。

## 三、当前问题

1. **每局重置**：`m_aStats` 在每局开始时重置，无法跨局统计
2. **无持久化**：统计数据仅存在于内存中
3. **缺少历史记录**：无法查看历史游戏表现
4. **缺少可视化**：没有图表展示统计趋势

---

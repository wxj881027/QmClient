# QmClient 观战导演模式功能报告

## 一、功能概述

导演模式是一种智能观战系统，自动切换观战目标，优先显示精彩画面。适用于视频制作、比赛直播等场景。

## 三、当前问题

1. **缺少自动切换逻辑**：需要手动切换观战目标
2. **缺少优先级判断**：无法自动选择精彩画面
3. **缺少画中画功能**：无法同时显示多个视角
4. **缺少平滑过渡**：切换目标时视角跳跃
5. **缺少录制集成**：无法与视频录制联动

## 四、实现方案

### 4.1 自动切换逻辑

#### 4.1.1 优先级系统

根据游戏事件计算每个玩家的"精彩度"分数：

```cpp
struct SDirectorScore
{
    int m_ClientID;
    float m_Score;
    float m_Decay;  // 衰减因子
};

// 评分因素
float CalculateDirectorScore(int ClientID)
{
    float Score = 0.0f;
    
    // 战斗相关
    Score += m_aStats[ClientID].m_Frags * 10.0f;      // 击杀
    Score += m_aStats[ClientID].m_FlagCaptures * 50.0f; // 夺旗
    
    // 位置相关
    if(IsNearFlag(ClientID)) Score += 20.0f;          // 靠近旗帜
    if(IsInCombat(ClientID)) Score += 15.0f;          // 战斗中
    
    // 速度相关
    Score += GetSpeed(ClientID) * 0.1f;               // 移动速度
    
    return Score;
}
```

#### 4.1.2 切换策略

| 优先级 | 事件 | 切换延迟 |
|--------|------|---------|
| 最高 | 旗帜捕获 | 立即 |
| 高 | 击杀事件 | 0.5s |
| 中 | 战斗中 | 1.0s |
| 低 | 普通移动 | 3.0s |

### 4.3 平滑过渡

#### 4.3.1 摄像机插值

```cpp
// 从当前视角平滑过渡到目标
vec2 CameraTransition(vec2 Current, vec2 Target, float Progress)
{
    // 使用缓动函数
    float t = EaseInOutCubic(Progress);
    return mix(Current, Target, t);
}
```

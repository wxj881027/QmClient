# QmClient 截图管理器功能报告

## 一、功能概述

截图管理器统一管理游戏截图，支持自动添加水印/时间戳/地图名，并提供一键分享功能。

## 三、当前问题

1. **缺少管理界面**：无法在游戏内浏览截图
2. **缺少水印功能**：无法自动添加水印
3. **缺少元数据**：截图不包含地图名、时间戳等信息
4. **缺少分享功能**：无法一键分享截图
5. **文件名不友好**：默认文件名不包含游戏信息

struct ScreenshotInfo

示例：Novice_20240419_153000_001.png

### 4.2 水印系统

#### 4.2.1 水印配置

```cpp
struct SWatermarkConfig
{
    bool m_Enabled;
    int m_Position;         // 位置（0=左上, 1=右上, 2=左下, 3=右下）
    float m_Opacity;        // 透明度
    bool m_ShowTimestamp;   // 显示时间戳
    bool m_ShowMapName;     // 显示地图名
    bool m_ShowServerName;  // 显示服务器名
    bool m_ShowPlayerName;  // 显示玩家名
    char m_aCustomText[64]; // 自定义文本
};
```

#### 4.2.2 水印渲染

```cpp
void RenderWatermark(CGfx *pGfx, const SWatermarkConfig &Config)
{
    char aWatermarkText[256];
    str_format(aWatermarkText, sizeof(aWatermarkText),
        "%s | %s | %s",
        Config.m_ShowTimestamp ? GetTimestamp() : "",
        Config.m_ShowMapName ? GetCurrentMap() : "",
        Config.m_aCustomText
    );
    
    // 渲染到截图
    pGfx->RenderText(aWatermarkText, Config.m_Position, Config.m_Opacity);
}
```

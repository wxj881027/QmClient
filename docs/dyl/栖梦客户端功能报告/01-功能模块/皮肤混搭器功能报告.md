# QmClient 皮肤混搭器功能报告

## 一、功能概述

皮肤混搭器（Skin Mixer）允许玩家将不同皮肤的身体部位（头、身体、手等）进行组合，创造独特的个性化外观。

## 三、当前问题

1. **无法混搭部位**：只能使用完整皮肤，无法组合不同皮肤的部位
2. **缺少预览**：无法实时预览混搭效果
3. **缺少保存**：混搭配置无法保存
4. **缺少分享**：无法分享混搭皮肤

## 四、实现方案

### 4.1 混搭数据结构

```cpp
struct SSkinMixer
{
    char m_aBodySkin[24];       // 身体来源皮肤
    char m_aHandsSkin[24];      // 手部来源皮肤
    char m_aFeetSkin[24];       // 脚部来源皮肤
    char m_aEyesSkin[24];       // 眼部来源皮肤
    char m_aMarkingSkin[24];    // 花纹来源皮肤
    char m_aDecorationSkin[24]; // 装饰来源皮肤
    
    // 各部位颜色覆盖（可选）
    bool m_OverrideColors[6];
    ColorRGBA m_CustomColors[6];
};
```

### 4.2 混搭渲染

#### 4.2.1 纹理合成

```cpp
IGraphics::CTextureHandle ComposeMixerSkin(const SSkinMixer &Mixer)
{
    // 创建临时渲染目标
    Graphics()->CreateRenderTarget(...);
    
    // 按部位从源皮肤纹理复制
    CopyBodyPart(Mixer.m_aBodySkin, BODY_REGION);
    CopyBodyPart(Mixer.m_aHandsSkin, HANDS_REGION);
    CopyBodyPart(Mixer.m_aFeetSkin, FEET_REGION);
    CopyBodyPart(Mixer.m_aEyesSkin, EYES_REGION);
    
    // 应用颜色覆盖
    if(Mixer.m_OverrideColors[BODY])
        ApplyColorTint(BODY_REGION, Mixer.m_CustomColors[BODY]);
    
    return Graphics()->GetRenderTargetTexture();
}
```

#### 4.2.2 缓存机制

```cpp
class CSkinMixerCache
{
    std::map<uint64_t, IGraphics::CTextureHandle> m_Cache;
    
    uint64_t ComputeHash(const SSkinMixer &Mixer);
    IGraphics::CTextureHandle GetOrCreate(const SSkinMixer &Mixer);
    void ClearCache();
};
```

### 4.4 保存与分享

#### 4.4.1 保存格式

```json
{
    "name": "My Custom Mix",
    "parts": {
        "body": "default",
        "hands": "ninja",
        "feet": "default",
        "eyes": "penguin",
        "marking": "camo",
        "decoration": "santa_hat"
    },
    "colors": {
        "body": [1.0, 0.5, 0.0, 1.0]
    }
}
```

#### 4.4.2 分享方式

1. **导出文件**：保存为 `.qmixin` 文件
2. **分享码**：生成短码，其他玩家可导入
3. **Workshop 上传**：上传到社区分享

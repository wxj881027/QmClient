# QmClient 聊天记录导出功能报告

## 一、功能概述

聊天记录导出功能允许玩家将游戏内聊天记录保存为文本或 JSON 文件，方便保存精彩对话或作为举报违规的证据。

---

## 三、当前问题

1. **无法导出**：聊天记录仅存在于内存中，无法持久化
2. **容量限制**：`MAX_LINES` 限制记录数量，旧消息会被覆盖
3. **缺少时间戳**：消息没有精确的时间戳（只有相对时间）
4. **缺少过滤**：无法按条件筛选导出内容

---

## 四、实现方案

### 4.1 导出格式

#### 4.1.1 文本格式（TXT）

```
=== QmClient Chat Log ===
Server: DDNet.tw - Novice
Date: 2024-04-19 15:30:00
=========================

[15:30:05] [All] Player1: Hello everyone!
[15:30:12] [Team] Player2: Let's go left
[15:30:45] [All] Player3: Nice shot!
...
```

#### 4.1.2 JSON 格式

```json
{
    "version": 1,
    "export_time": "2024-04-19T15:30:00Z",
    "server": {
        "name": "DDNet.tw - Novice",
        "address": "ddnet.tw:8303"
    },
    "messages": [
        {
            "time": "2024-04-19T15:30:05Z",
            "channel": "all",
            "sender": {
                "id": 5,
                "name": "Player1"
            },
            "content": "Hello everyone!"
        }
    ]
}
```

### 4.2 导出选项

```cpp
struct SExportOptions
{
    bool m_IncludeTimestamp;    // 包含时间戳
    bool m_IncludeServerInfo;   // 包含服务器信息
    int m_ChannelFilter;        // 频道过滤（0=全部, 1=全体, 2=队伍, 3=私聊）
    int m_TimeRange;            // 时间范围（0=全部, 1=最近1小时, 2=最近10分钟）
    int m_Format;               // 格式（0=TXT, 1=JSON）
};
```

### 4.3 配置变量

| 变量 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `cl_chat_export_format` | INT | 0 | 导出格式（0=TXT, 1=JSON） |
| `cl_chat_export_timestamp` | INT | 1 | 包含时间戳 |
| `cl_chat_export_server_info` | INT | 1 | 包含服务器信息 |
| `cl_chat_auto_save` | INT | 0 | 自动保存聊天记录 |
| `cl_chat_max_lines` | INT | 1000 | 最大保存行数 |

### 4.4 导出触发方式

1. **控制台命令**：`export_chat [filename]`
2. **UI 按钮**：聊天框设置中添加导出按钮
3. **自动保存**：退出服务器时自动保存


# QmClient 功能头脑风暴报告

## 一、概述

本报告基于对 QmClient 现有功能报告的全面分析，从功能缺口、体验提升、长期规划、全新创意、技术债务、社区生态六个维度进行头脑风暴，为后续开发提供灵感参考。

**注意**：本报告中的条目为创意提案，优先级和可行性需结合具体报告进一步评估。部分条目已转化为独立功能报告或 todo 项。

---

## 二、高优先级 — 报告中已明确缺失的核心功能

这些是报告中反复提到"未实现"或"需要完善"的功能，用户感知最强：

| # | 功能 | 灵感来源 | 当前状态 |
|---|------|---------|---------|
| 1 | **Speedrun Timer（速通倒计时器）** | BC端移植报告 | BC 仅 80 行核心代码，无依赖；QmClient 有正计时但缺倒计时+时间耗尽自动 Kill；→ 已加入 todo 9.3 |
| 2 | **Auto Team Lock（自动队伍锁定）** | BC端移植报告 | BC 仅 55 行代码，最简单的移植功能，DDRace 玩家刚需；→ 已加入 todo 9.5 |
| 3 | **出站翻译启用** | 翻译功能报告 | `TryTranslateOutgoingChat()` 逻辑已实现，仅差取消 `SendChatQueued()` 中的一行注释；→ 已加入 todo 7.2 |
| 4 | **智谱AI翻译后端** | 翻译功能报告 | 免费额度、LLM 翻译质量高、可通过 Prompt 优化游戏术语，报告已有完整实现方案；→ 已加入 todo 7.3 |
| 5 | **聊天框翻译按钮** | 聊天框翻译按钮报告 | 当前翻译只能靠命令行触发，需要从零添加聊天框鼠标交互基础设施；→ 已加入 todo 7.1 |
| 6 | **NetGraph 网络延迟历史图表** | 网络卡顿可视化报告 | 引擎已有 `m_FpsGraph`（FPS 图表），但没有对应的网络延迟图表，无法直观看到卡顿趋势；→ 已加入 todo 5.2 |

## 三、中优先级 — 体验提升与系统完善

这些功能能显著提升用户体验，但实现难度中等：

| # | 功能 | 灵感来源 | 当前状态 |
|---|------|---------|---------|
| 7 | **统一 UI 主题/样式系统** | UI和动画报告 | 当前颜色/样式分散在数十个配置变量中，`SUiStyle` 仅覆盖布局不含视觉属性，缺少 Theme/Style 集中管理；→ 已加入 todo 1.4 |
| 8 | **合并三套重复动画辅助函数** | UI和动画报告 | `ResolveAnimatedLayoutValueEx`/`ResolveUiAnimValue`/`ResolveAnimatedValue` 功能几乎相同，应统一；→ 已加入 todo 1.2 |
| 9 | **Chat Binds UI 增强** | Bind指令系统报告 | 当前只能编辑默认绑定名称，无法添加自定义绑定、修改命令内容、删除绑定；→ 已加入 todo 8.2 |
| 10 | **Tab 自动补全集成 BindChat** | Bind指令系统报告 | `ChatDoAutocomplete` 方法已实现但未被调用，聊天框 Tab 补全只处理服务器命令和玩家名；→ 已加入 todo 8.3 |
| 11 | **3D 粒子效果移植** | BC端移植报告 | 实际难度比预期低（纯线框渲染+简单透视投影，非真正3D引擎），约760行独立代码；→ 已加入 todo 9.2 |
| 12 | **Media Background 媒体背景** | BC端移植报告 | FFmpeg 已有依赖，支持视频/动画/静态图片背景，约1650行代码；→ 已加入 todo 9.4 |
| 13 | **连接质量分级显示** | 网络卡顿可视化报告 | 当前 `ConnectionProblems()` 只有"有/无"二值判断，应改为多级：正常/偏高/严重/断开；→ 已加入 todo 5.1 |
| 14 | **丢包率 + Jitter 显示** | 网络卡顿可视化报告 | `NETSTATS` 只有收发包计数，缺丢包/Jitter 指标；引擎内部有 `m_AutoMarginLatencyJitterMs` 但未暴露给 UI；→ 已加入 todo 5.1 |
| 15 | **创建 QmClient 扩展语言文件** | 汉化完善报告 | `data/qmclient/languages/` 目录不存在，双层加载机制的第二层从未生效；→ 已加入 todo 12.1 |
| 16 | **修复 ~117 处硬编码中文** | 汉化完善报告 | `Localize("中文")` 约50处 + 直接中文字符串约67处，严重影响国际化；→ 已加入 todo 12.2 |
